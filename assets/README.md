# Agions Profile Assets

## 需要的资源文件

请将以下文件添加到 `assets` 目录：

1. **banner.png** - 个人主页横幅图片
   - 推荐尺寸: 1280x640 像素
   - 可以使用在线工具生成: https://banner. Make.com

2. **wechat-qrcode.png** - 微信公众号二维码
   - 推荐尺寸: 300x300 像素
   - 请到微信公众号后台下载官方二维码图片
